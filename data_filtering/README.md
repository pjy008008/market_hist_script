# 데이터 필터링과 봉 집계

새 통합 파이프라인은 Alpaca SIP 30분봉을 다음과 같이 처리합니다.

1. `filter_regular_session.py`로 XNYS 정규장 봉만 선택
2. `resample_sip_5min.py`의 공통 집계 함수로 1시간·4시간·일봉 생성
3. 30분봉은 폐기하고 요청한 세 주기만 저장

XNYS 캘린더를 사용하므로 휴장일, 조기 폐장, 서머타임을 반영합니다. 모든 봉은 미국 정규장 개장 시각에 정렬됩니다.

집계 규칙:

- 시가: 첫 원본 봉의 시가
- 고가: 원본 봉 고가의 최댓값
- 저가: 원본 봉 저가의 최솟값
- 종가: 마지막 원본 봉의 종가
- 거래량·체결 수: 합계
- VWAP: 거래량 가중 평균
- `source_minutes`: 포함된 30분 입력 구간의 합계

기존 1분봉 필터링과 5·15분봉 독립 CLI는 보조 용도로 유지되지만 `daily_pipeline.py`에서는 실행하지 않습니다.
